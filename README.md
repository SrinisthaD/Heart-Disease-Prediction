# Heart-Disease-Prediction
## 📌 Overview
This project predicts the likelihood of heart disease using machine learning classification algorithms. The complete workflow includes exploratory data analysis, outlier detection, data preprocessing, and model comparison.

## 🔧 Libraries Used
- Pandas, NumPy (Data manipulation)
- Matplotlib, Seaborn (Visualization)
- Scikit-learn (Machine Learning models)

## 📈 Steps Performed
1. Exploratory Data Analysis (EDA)
2. Outlier detection and handling using IQR method
3. Data visualization (boxplots, correlation matrix)
4. Feature scaling using StandardScaler
5. Train-test split
6. Model building and comparison:
   - Logistic Regression
   - Linear Regression
   - K-Nearest Neighbors (KNN)
   - Support Vector Machine (SVM)
   - Random Forest Classifier
7. Model evaluation using accuracy scores
